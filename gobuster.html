GoBuster

GoBuster é uma ferramenta de força bruta para descobrir arquivos e diretórios em servidores web.

Instalação

Para instalar o GoBuster, você pode usar o seguinte comando:

go get -u github.com/OJ/gobuster/v3

Uso Básico

O uso básico do GoBuster é o seguinte:

gobuster dir -u  -w

Substitua pela URL do alvo e pelo caminho para sua lista de palavras.

Exemplos de Uso:

Descoberta de Diretórios: 

gobuster dir -u http://exemplo.com -w /caminho/para/wordlist.txt

Descoberta de Arquivos: 

gobuster dir -u http://exemplo.com -w /caminho/para/wordlist.txt -x php,html

Opções Comuns

  • -u: Especifica a URL do alvo.
  • -w: Especifica o caminho para a lista de palavras.
  • -x: Especifica as extensões de arquivo a serem testadas.
  • -t: Define o número de threads para usar.
  • -o: Especifica o arquivo de saída para salvar os resultados.

Considerações Finais

GoBuster é uma ferramenta poderosa para descobrir arquivos e diretórios ocultos em servidores web. Use-a com responsabilidade e sempre obtenha permissão antes de realizar testes de penetração em qualquer sistema.
